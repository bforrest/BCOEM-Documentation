## When using Table Planning Mode, Pull Sheets are Not Available

Indeed, pullsheets are **not** available when in Table Planning Mode - this is to prevent any confusion since pullsheets are only for entries that are received, checked-in, and ready for judging. Otherwise, stewards/staff/cellar people could possibly be chasing down ghosts. Simply switching to Tables Competition Mode, like you did, will provide the appropriate pullsheet icons/links - providing entries have been marked as received in the application.
